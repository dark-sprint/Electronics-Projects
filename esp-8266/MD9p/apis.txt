# API para obtener información geográfica y de IP
# https://ipapi.co/json - Proporciona información sobre la ubicación y detalles de una dirección IP.
# https://ipinfo.io/json - Similar a ipapi, ofrece datos sobre la ubicación, ISP y más de una IP.
# https://rdap.db.ripe.net/ip/<IP> - Proporciona información de registro sobre una dirección IP específica desde la base de datos RIPE.
# http://api.positionstack.com/v1/reverse?access_key={API_KEY}&query={lat},{lon} - Realiza una búsqueda inversa de geocodificación para obtener una dirección a partir de coordenadas geográficas.

# API para obtener datos meteorológicos
# curl "https://api.openweathermap.org/data/2.5/weather?lat=40.7128&lon=-74.0060&appid=YOUR_API_KEY&units=metric" - Proporciona información meteorológica actual para una ubicación específica usando coordenadas.

# API para generar datos aleatorios
# https://randomuser.me/api/ - Genera datos de usuario aleatorios, como nombre, dirección, etc.
# https://jsonplaceholder.typicode.com/users - Proporciona datos de usuarios ficticios para pruebas y desarrollo.
# https://api.coingecko.com/api/v3/coins/bitcoin - Proporciona información sobre el precio y datos de la criptomoneda Bitcoin.
# https://api.qrserver.com/v1/create-qr-code/?data=HelloWorld - Genera un código QR a partir de un texto proporcionado.
# https://fakerapi.it/api/v1/persons?quantity=1&locale=es_ES - Genera datos ficticios de personas en español.
# https://randomuser.me/api/?nat=es - Genera datos de usuario aleatorios, filtrando por nacionalidad española.

# API para datos de aviación
# https://opensky-network.org/api/states/all - Proporciona información sobre el estado actual de los vuelos en el espacio aéreo.

# API para direcciones y rutas
# https://maps.googleapis.com/maps/api/directions/json?origin=Place_A&destination=Place_B&traffic_model=best_guess&departure_time=now&key=YOUR_API_KEY - Proporciona direcciones y rutas entre dos lugares, considerando el tráfico.

# API para datos de salud y enfermedades
# https://disease.sh/v3/covid-19/all - Proporciona datos actualizados sobre la pandemia de COVID-19 a nivel mundial.
# https://api.nasa.gov/mars-photos/api/v1/rovers/curiosity/photos - Proporciona fotos tomadas por el rover Curiosity en Marte.
# https://earthquake.usgs.gov/fdsnws/event/1/query - Proporciona información sobre terremotos recientes y sus características.

# API para chatbots y generación de texto
# https://api.openai.com/v1/completions - Proporciona acceso a modelos de lenguaje para generar texto y respuestas.
# https://botpress.com/docs/api-reference/chat-api/introduction - Proporciona una API para interactuar con chatbots desarrollados en Botpress.
# Wit.ai (Gratis e ilimitado) - https://wit.ai/ - Proporciona herramientas para construir aplicaciones que entienden el lenguaje natural.

# API para vulnerabilidades de seguridad
# https://nvd.nist.gov/vuln/data-feeds#JSON_Vulnerability_Feeds - Proporciona feeds de datos sobre vulnerabilidades de seguridad en formato JSON.
# https://nvd.nist.gov/developers/vulnerabilities - Documentación para acceder a datos sobre vulnerabilidades.
# https://services.nvd.nist.gov/rest/json/cves/2.0 - Proporciona acceso a datos de CVEs (Common Vulnerabilities and Exposures) en formato JSON.
# https://nvd.nist.gov/developers/products - Información sobre productos y servicios relacionados con la gestión de vulnerabilidades.

# encender led 

#define LED_BUILTIN 2  // GPIO2 (D4)

void setup() {
  pinMode(LED_BUILTIN, OUTPUT);
  digitalWrite(LED_BUILTIN, HIGH);  // Apaga el LED (activo en bajo)
}

void loop() {
  digitalWrite(LED_BUILTIN, LOW);   // Enciende
  delay(500);
  digitalWrite(LED_BUILTIN, HIGH);  // Apaga
  delay(500);
}
